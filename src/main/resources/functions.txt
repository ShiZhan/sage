Usage: sage [-hipmg] <options> [input file]

 -h   print this message

 -i   [edges] <options>, input edge list as 2 columns csv or console, comments start with '#'.
      --shard  [shard], the number of edge list shards, should be power of 2.

 -p   [algorithm] [imported edges] <options>, process imported graph with designated algorithm
      --shard  [shard], the number of edge list shards, should be power of 2.
 
      supported algorithms:
      BFS:root(Long),  E.g.: "bfs:0"
      DFS:root(Long),  E.g.: "dfs:0"
      SSSP:root(Long), E.g.: "sssp:0"
      PAGERANK,        E.g.: "pagerank"

 -m   [vertex map] [edges], remap vertex id by given map.
      vertex map is stored in Long sequence, map Long value to corresponding line number.

 -g   [generator] parameters are splitted by ':'.
      RMAT:scale(Int):degree(Int), E.g.: rmat:8:8
      ER:scale(Int):ratio(Double), E.g.: er:8:0.1
